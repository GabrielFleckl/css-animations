## 💻 Projeto

Neste projeto aprendi a usar animações e com isso fazer um formulário animado, com JS puro e usando CSS Animation, desenvolvido com 8 propriedades de animation sendo elas:

- animation-name: animationname;
- animation-duration: 2s;
- animation-delay: 3s;
- animation-fill-mode: none;
- animation-play-state: running;
- animation-timing-function: ease;
- animation-direction: reverse;
- animation-iteration-count: infinite;
